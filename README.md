# Space Gen

Space Gen is a space-themed web project that includes a solar system simulation, individual planet viewer pages, user signup/login pages, merch listing, and checkout flow.

## Features

- Interactive solar system simulation
- Clickable planets with detailed planet information
- 3D planet viewer using Three.js
- Signup and login pages with basic JavaScript validation
- Merch product page
- Checkout page with basic payment form validation
- Animated star background
- Responsive space-themed UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Three.js

## Pages

- `index.html` - Home page
- `signup.html` - Signup page
- `login.html` - Login page
- `dashboard.html` - Dashboard page
- `simulation.html` - Solar system simulation
- `planets.html` - Planet detail viewer
- `merch.html` - Merchandise page
- `checkout.html` - Checkout page

# Project Purpose
This project was created to practice frontend development using HTML, CSS, JavaScript, and Three.js. It focuses on building an interactive space-themed website with animations, validation, and 3D visuals.
